What value of \(x\) satisfies the equation?

$$2x+3=5x$$

Choose 1 Answer:

Approach 1

Start by subtracting \(2x\) from both sides, then divide by 3.

$$ 2x+3=5x$$ $$3=3x$$ $$1=x$$

Approach 2

For multiple choice questions, we can always plug in the answer choices into the equation to see if the value(s) satisfy the given equation. You should also check your work using this method.

$$2x+3=5x \text { ; } x=1$$ $$2(1)+3=5(1)$$ $$2+3=5$$ $$5=5 \ \ ✓$$
